# Java Fundamentals Exercises

This repository contains basic Java exercises created for learning and practice.

## Topics Covered

### Loops
Examples of for, while, and do-while loops.

### Arrays
Basic array operations: sum, search, average.

### Classes
Simple Java classes and object usage.

### Inheritance
Examples using inheritance and polymorphism.

## How to Run
- Export this project in Eclipse / IntelliJ
- Compile and run the Java files per folder

## Notes
Projects are beginner level, created for learning purposes.
