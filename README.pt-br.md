# Exercícios de Fundamentos Java

Este repositório contém exercícios básicos em Java, desenvolvidos para estudo e prática.

## Tópicos Abordados

### Laços
Exemplos utilizando laços `for`, `while` e `do-while`.

### Arrays
Operações básicas com arrays, como soma, média e busca de valores.

### Classes
Criação de classes simples, atributos, métodos e instância de objetos.

### Herança
Exemplos básicos de herança e sobrescrita de métodos.

## Como Executar
1. Importe o projeto no Eclipse ou IntelliJ IDEA.
2. Navegue até o pacote desejado.
3. Execute a classe Java que contém o método `main`.

## Observações
Este repositório possui exercícios em nível iniciante e será atualizado conforme o avanço nos estudos de Java.
